# Hi3861 WiFi-IoT LED控制示例项目

## 项目简介

本项目是基于华为Hi3861 WiFi芯片的OpenHarmony物联网应用示例，实现了一个简单的LED控制功能。项目演示了如何使用Hi3861开发板的GPIO接口控制LED灯的开关和闪烁效果。

## 主要功能

- **LED控制**：支持LED灯的开启、关闭和闪烁三种状态
- **多任务处理**：使用CMSIS-RTOS2实时操作系统创建独立的LED控制任务
- **GPIO操作**：演示了Hi3861 GPIO的初始化和输出控制

## 硬件要求

- 普中Hi3861开发板
- LED灯（连接至GPIO2引脚）
